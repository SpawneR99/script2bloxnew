Script Troubleshooting Guide — Common Issues & Solutions

GUI Not Appearing or Appearing Off-Screen

GUIs sometimes spawn off the visible area, especially on unusual display resolutions. Most scripts bind a toggle key, usually Home or right Shift on PC, or a floating button on mobile. Press the toggle to force the GUI back into view.

Features Not Working Despite GUI Loading

GUI loads but toggles do nothing? The script probably needs a patch for the latest game update. Check RoCrack for a newer version. Also make sure you waited for the game to fully load before injecting... executing during a loading screen breaks most scripts.

Game Crashing After Script Execution

Crashes usually mean a memory conflict between the script and a specific game mechanic, or you've got too many features running on a device that can't handle the load. Disable heavy features like ESP overlays, enable things one at a time to find the culprit, and check that your device meets the script's minimum specs.

Script Working Inconsistently

Works sometimes, fails other times? Timing is almost always the cause. Inject and execute only after the game is fully loaded, never during loading screens or server transitions. If the issue persists, hop to a different server. Some servers just behave differently.
