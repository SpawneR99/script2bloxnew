Script Troubleshooting Guide — Common Issues & Solutions

If it still won't load, check that injection succeeded, re-inject Delta, confirm you're on the latest executor version, and verify the script supports the current game patch.

GUI Not Appearing or Appearing Off-Screen

GUIs occasionally spawn outside visible bounds, especially on unusual display resolutions. Most scripts have a keybind to toggle the GUI, usually Home or Right Shift on PC, or a floating button on mobile. Press those before concluding the script is broken.

Features Not Working Despite GUI Loading

If the GUI renders but features don't respond when toggled, the script probably needs a patch for the latest game version. Check RoCrack for updates. Also make sure you let the game finish loading completely before you injected... executing mid-load is the most common cause of partial feature failures.

Game Crashing After Script Execution

Crashes typically trace to script conflicts with game mechanics, or too many features enabled at once on a device that can't handle the memory hooking overhead. Disable resource-heavy features. Enable them one at a time to find the culprit. And verify your device actually meets the script's requirements.

Script Working Inconsistently

Inconsistent results almost always come down to execution timing. Inject after the game is fully loaded, not during transitions or loading screens. Server differences can also play a role, so try swapping to another server if features work on one but fail on the next.
