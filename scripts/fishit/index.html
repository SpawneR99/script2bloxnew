GUIs can spawn off-screen, especially on unusual display resolutions or when switching between fullscreen and windowed mode. Most scripts bind a toggle key, usually Home or right Shift on PC, or a floating icon on mobile. Tap or press it to bring the GUI back into view.

Features Not Working Despite GUI Loading

The GUI shows up fine but toggles do nothing? The script likely needs a patch for the latest game update. Check RoCrack for a newer version. Also make sure you let the game fully load before injecting... running scripts during loading screens is the most common cause of this problem.

Game Crashing After Script Execution

Crashes usually mean a conflict between the script and a specific game mechanic, or too many features running at once on hardware that cannot handle the load. Turn off heavy features, re-enable them one by one to find the culprit, and confirm your device meets the script's minimum requirements.

Script Working Inconsistently

Sometimes works, sometimes does not? Timing is almost always the issue. Only inject and execute after the game world is completely loaded. Server-side conditions can vary too, so try hopping to a different server if one seems problematic.
