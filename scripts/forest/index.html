Crashes usually mean too many features are running simultaneously, or a specific feature conflicts with Forest's game mechanics. Turn off heavy features, re-enable them one at a time to find the culprit, and verify your device has enough RAM for the script you're running.

Script Working Inconsistently

Inconsistency almost always comes from execution timing. Inject after the game is completely loaded, not during loading screens or transitions. Server differences can also be a factor. If features work on one server but not another, just hop.
